# covid19-system

#### Description
基于Python的全国疫情监控系统
基于 Python+Flask+Echarts 技术，疫情数据收集通过网络爬虫技术爬取实时疫情、网站的搭建是基于Flask 框架，在此基础上搭配 Echarts 来将数据库中的数据映射成图表形式，实现数据可视化，同时利用 Ajax 技术局部刷新页面完成数据不断更新。
